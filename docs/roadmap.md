# Roadmap

## Phase 1 - MVP
- Set up Flask backend
- Set up React frontend with Vite
- Add file upload
- Run static analysis on Python files
- Generate a code health score
- Display results in a dashboard

## Phase 2 - UX Improvements
- Add charts and stronger visual hierarchy
- Improve issue explanations
- Add severity filters
- Improve upload experience and validation

## Phase 3 - Differentiation
- Add AI-generated explanations
- Add trend tracking for repeat scans
- Add complexity heatmap visualization

## Phase 4 - Product Signals
- Add GitHub integration
- Save scan history
- Add shareable dashboard links
