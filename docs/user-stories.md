# User Stories

## Developer
As a developer, I want to upload my code and quickly see which files need attention so that I can prioritize refactoring.

## Product Manager
As a product manager, I want to view a simple health score and issue summary so that I can understand engineering risk without needing to read code.

## Student Engineer
As a student engineer, I want clear explanations for flagged issues so that I can learn how to improve my code.
